# ProjetoFinalHtml
Projeto de um site para uma faculdade ficticia com conceitos básicos de HTML.

Foi utilizados os seguintes conceitos básicos:
- Contrução de links
- Tabelas
- Cabeçalhos
- Parágrafos
- Formatações de textos
- Listas ordenadas e não ordenadas
- Imagens
- Formulários
- Caracteres especiais
